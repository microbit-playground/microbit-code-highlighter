Microbit Python Code Highlighting with Prism JS

(Slightly) modified themes and python parser for PrismJS.

To use on website, add these to your <HEAD> block:

<script src="prism.js"></script>
<link href="prism-coy.css" rel="stylesheet" type="text/css" id="stylelink">

To call the code highlighter within your HTML document enclose the code with PRE & CODE tags. Eg:

<pre><code class="language-pythonmicrobit">

from microbit import *
import random

</code></pre>

Examples of it in use at microbit playground.
